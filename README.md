# MoneyTracker (家計簿アプリ)

シンプルで使いやすい家計簿アプリケーションです。日々の収支を簡単に記録し、無駄遣いを減らすお手伝いをします。

## ✨ 主な機能 (主要功能)

*   **収支の記録**: 収入と支出を素早く入力できます。
*   **カテゴリー管理**: 食費、交通費、家賃など、カテゴリーごとに分類可能です。
*   **グラフ表示**: 月ごとの支出推移をグラフで可視化します。
*   **データの保存**: データはローカルデータベースに安全に保存されます。
*   **履歴の確認**: 過去の取引履歴を一覧で確認・編集・削除できます。
